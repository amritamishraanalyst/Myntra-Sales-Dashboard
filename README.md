# Myntra-Sales-Analysis-Dashboard

## 1. Project Title / Headline

🛍️ **Myntra Sales Analysis Dashboard**

An interactive Power BI dashboard designed to analyse Myntra's sales performance, revenue, orders, discounts, brands, product categories, and state-wise sales distribution.


## 2. Short Description / Purpose

The Myntra Sales Analysis Dashboard provides a clear overview of sales performance across different brands, categories, products, states, and reporting periods.

SQL was used for querying, cleaning, validating, and analysing the dataset. Microsoft Excel was used for initial data preparation, while Power BI was used for data modelling, DAX calculations, interactive visualisation, and dashboard development.

The dashboard allows users to filter the results by year, customer category, brand, and product sub-category.


## 3. Tech Stack

- 🗄️ **SQL** – Data querying, cleaning, validation, and analysis
- 📊 **Power BI Desktop** – Dashboard development and visualisation
- 📂 **Microsoft Excel** – Initial data cleaning and preparation
- 🧠 **DAX** – Creation of calculated measures and KPIs
- 🔄 **Power Query** – Data transformation and formatting


## 4. Data Source

**Source: Kaggle**

The dataset contains Myntra sales information, including:

- Order details
- Sales amount
- Revenue
- Discount percentage
- Product categories
- Product sub-categories
- Brand names
- Customer categories
- State-wise sales
- Year and month details


## 5. Dashboard Features

### Business Problem

Large retail datasets can make it difficult to quickly identify sales trends, top-performing brands, popular product categories, regional performance, and the impact of discounts on sales.

### Goal of the Dashboard

The dashboard was developed to:

- Monitor total orders, revenue, and sales
- Compare performance across 2021, 2022, and 2023
- Identify top-performing brands and products
- Analyse category and sub-category sales
- Evaluate state-wise revenue
- Understand sales patterns across different days
- Examine the relationship between sales and discounts
- Support data-driven sales and inventory decisions
